🃏 Collection de Cartes

Collectionne des cartes sur tes thèmes préférés !

  • Piocher des cartes : Clique sur "🎁 Piocher une carte" pour obtenir de nouvelles cartes aléatoires
  • Limite quotidienne : Tu peux piocher 5 cartes par jour
  • Raretés :
    • 🤍 Commune (60%)
    • 💙 Rare (25%)
    • 💚 Très Rare (10%)
    • 💛 Épique (4%)
    • ❤️ Légendaire (1%)
  • Thèmes : Tu peux avoir entre 3 et 10 thèmes actifs dans ta collection
  • Filtres : Trie tes cartes par rareté ou ordre alphabétique

🪙 Crédits

Les crédits te permettent de piocher des cartes. 1 crédit = 1 carte à piocher.

  • Gagner des crédits : Joue quotidiennement aux mini-jeux pour gagner des crédits
  • Consultation : Ton solde de crédits est affiché en haut à gauche dans la navigation

🎮 Mini-Jeux

🎓 Bonus Mathématiques

Résous des opérations mathématiques pour gagner des cartes bonus.

  • ➕ Addition : 1 carte par bonne réponse
  • ➖ Soustraction : 2 cartes par bonne réponse
  • ✖️ Multiplication : 5 cartes par bonne réponse
  • 3 essais par type d'opération par jour

🧭 Déplacement sur Quadrillage

Programme le chemin du personnage pour atteindre la cible en visitant un maximum de cases.

  • Utilise les flèches pour programmer tes mouvements
  • Objectif : Atteindre la cible en visitant le plus de cases possibles
  • Règle importante : Ne repasse JAMAIS sur une case déjà visitée (sinon 1 point seulement !)
  • Évite les obstacles (🔴) et les murs
  • Ton score = nombre de cases uniques visitées
  • Partie de 10 niveaux, difficulté croissante
  • Touche Entrée pour valider

🔍 Mots Mêlés

Trouve tous les mots cachés dans la grille.

  • Clique et glisse sur les lettres pour sélectionner un mot
  • Les mots peuvent être dans toutes les directions (horizontal, vertical, diagonal)
  • Trouve tous les mots pour terminer le niveau

🧩 Sudoku

Remplis la grille avec les chiffres de 1 à 9.

  • Chaque ligne, colonne et carré 3x3 doit contenir tous les chiffres de 1 à 9
  • Clique sur une case puis sur un chiffre pour le placer
  • Utilise l'indice si tu es bloqué (réduit la récompense)

🔢 Séquence de Nombres

Trouve le nombre manquant dans la séquence.

  • Analyse la suite logique des nombres
  • Entre le nombre qui complète la séquence
  • Difficulté croissante au fil des niveaux

🔐 Déchiffrage

Décode le message secret en utilisant le code César.

  • Utilise le décalage indiqué pour déchiffrer le message
  • Exemple : avec décalage 3, D devient A
  • Entre le message décodé pour valider

🕐 Lecture de l'Heure

Lis l'heure affichée sur l'horloge.

  • Observe la position des aiguilles
  • Entre l'heure au format HH:MM
  • Attention aux horloges analogiques et numériques

👤 Profil

Consulte tes statistiques et personnalise ton compte.

  • Statistiques : Nombre de cartes, crédits, etc.
  • Thèmes : Gère tes thèmes de collection (min 3, max 10)
  • Historique : Voir tes dernières activités

💡 Astuces

  • Joue quotidiennement aux mini-jeux pour accumuler des crédits
  • Varie les types d'opérations mathématiques pour maximiser tes gains
  • N'utilise les indices qu'en dernier recours pour garder plus de récompenses
  • Sur Déplacement sur Quadrillage, planifie bien ton chemin avant de valider
  • Consulte ton profil pour suivre ta progression
